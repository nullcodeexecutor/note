**IDEA 快捷键**
****

| 快捷键 | 功能 |
|---------------+------------------|
| **ctrl + x** | 剪切行 |
| **ctrl + d** | 复制当行到下一行，光标会移动到下一行 |
| **ctrl + ;** | 代码补全，自己修改后的，默认的好像是ctrl+j |
| **ctrl + /** | 单行注释 | 
| **ctrl + shift + /** | 多行注释 |
| **ctrl + shift + 上/下** | 移动代码 |
| **alt+ shift + 上/下** | 移动代码 |
| **ctrl + alt + l** | 格式化 |
| **ctrl + h** | 显示类继承结构 |
| **ctrl + shift + n** | 查找文件 |
| **ctrl + n** | 查找类 |
| **ctrl + 上/下** | 代码移动，光标仍停留在当行 |
| **art + insert** | 插入代码，比如get set 继承方法 |
| **art + enter** | 光标放在有错误的代码上时，键入该命令可以修复代码 |
| **home** | 回到行首，即第一个字符前 |
| **end** | 到行末，即最后一个字符后 |
| **art + f8** | debug时执行表达式 |
| **shift + f6** | rename |
| **ctrl + alt + t** | 选中代码时，可以在代码外层套for try if 等 |
| **ctrl + shift + +/-** | 展开代码/折叠代码 |
| **alt + F7** | 查看引用 |
